Simulation de réseaux d'opérateurs en présence d'un mécanisme de réputation Readme file
---------------------------------------------------------------------------------------


Afin de pouvoir modifier la formule de modification de la réputation, vous pourrez l'atteindre
dans les méthodes setRepSuccess et setRepFailed de la classe Operator.

Afin de pouvoir modifier la formule de probabilité d'échec, vous pourrez l'atteindre
dans la méthode failChance de la classe Operator.
